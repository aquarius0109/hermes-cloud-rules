## 集群目录命名规范

每个集群上的工作目录必须遵循统一命名：

```
/work/home/aquarius0109/
├── papers/                            # 按论文组织
│   ├── paper1/                        # 论文编号
│   │   ├── DFT/                       # 计算类型
│   │   │   ├── Kstruvite_relax/       # 具体任务
│   │   │   └── Mg_complex binding/    # 具体任务
│   │   ├── MD/
│   │   └── NEB/
│   ├── paper2/
│   └── paper_MPC_阴离子调控/
├── shared/                            # 共享资源
│   ├── pseudopotentials/
│   └── scripts/
└── scratch/                           # 临时文件（定期清理）
```

**命名规则：**
- 论文目录：`paper{N}` 或 `paper_{简称}`（如 `paper_MPC_阴离子调控`）
- 计算目录：`{体系}_{方法}_{参数}`（如 `Kstruvite_PBE_relax`）
- 禁止空格，用下划线
- 日期后缀可选：`_{YYYYMMDD}`
