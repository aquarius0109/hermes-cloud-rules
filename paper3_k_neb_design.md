## 一、定位与目标

### 科学问题
K-鸟粪石（MgKPO₄·6H₂O）中K⁺的微观迁移机制——迁移路径、能垒大小、掺杂调控效应。

### 输出
- K⁺沿[001]和[100]方向的迁移能垒曲线
- 本征+Co/Cu/Fe掺杂的能垒变化
- SI：Bader电荷分析、COHP成键分析
- 可与旧方案的UFF-MD结果对比，展示DFT精度的重要性

### 与三篇论文的关系
| 论文 | 尺度 | 本文的输入/输出 |
|------|------|----------------|
| #1 DFT | 电子尺度（带隙调控） | 掺杂策略 + 结构模板 |

| **#3（本文）** | **介观尺度 → DFT-NEB** | **K⁺迁移能垒 → 用于后续多晶MD电导率模型** |

## 二、K-鸟粪石结构基础

### 晶体学
- **空间群**：Pmn2₁ (#31)
- **单胞**：a=6.827, b=6.035, c=11.033 Å（QE-PBE+D3弛豫后）
- **2×1×1超胞**：a=13.654, b=6.035, c=11.033 Å, 100原子

### K⁺亚晶格（2×1×1超胞）

K⁺在超胞中排成**两层交错结构**：

```
z≈11.0 Å (顶层):    K₁(x≈0, y≈2.5)    K₃(x≈6.8, y≈2.5)
z≈5.5 Å  (中层):         K₂(x≈3.4, y≈3.6)     K₄(x≈10.3, y≈3.6)
```

K位点间距（最小镜像）：
| 路径 | 距离 | 方向 |
|------|------|------|
| K₁↔K₂ | 6.58 Å | [001]方向 + 侧向 |
| K₂↔K₃ | 6.58 Å | [001]方向 + 侧向 |
| K₁↔K₃ | 6.83 Å | [100]方向（纯a轴） |
| K₂↔K₄ | 6.83 Å | [100]方向（纯a轴） |

## 三、NEB计算方案

### 3.1 模型设置

采用**空位机制（vacancy mechanism）**：
- **IS**：K₂, K₃, K₄位点被K⁺占据，K₁为空位（3 K原子/超胞）
- **FS**：K₁, K₃, K₄位点被K⁺占据，K₂为空位（3 K原子/超胞）
- K⁺从K₂迁移到K₁

**弛豫策略**（关键修正——原方案此步缺失）：
1. IS结构 → 取自QE弛豫后结构，移除K₁ → 再做vc-relax确认局域极小
2. FS结构 → 取IS结构，将K⁺从K₂移到K₁（**使用最小镜像：x≈0，而非x=13.66**）→ 做vc-relax得到平衡FS
3. 用弛豫好的IS和FS作为NEB端点

### 3.2 计算参数

| 参数 | [001]方向 | [100]方向 |
|------|-----------|-----------|
| 迁移路径 | K₂(x=3.4,z=5.5) → K₁(x=0,z=11.0) | K₂(x=3.4,z=5.5) → K₄(x=10.3,z=5.5) |
| 迁移距离 | 6.58 Å | 6.83 Å |
| NEB方法 | CI-NEB | CI-NEB |
| Replicas | 7 | 7 |
| 软件 | CP2K 2024.1 | CP2K 2024.1 |
| 集群 | 集群6（武汉） | 集群6（武汉） |
| 泛函 | PBE+D3(BJ) | PBE+D3(BJ) |
| 基组 | DZVP-MOLOPT-SR-GTH | DZVP-MOLOPT-SR-GTH |
| 芯势 | GTH-PBE | GTH-PBE |
| SCF方法 | OT（轨道变换） | OT（轨道变换） |
| EPS_SCF | 1E-4（足够NEB收敛） | 1E-4 |
| 自旋极化 | ON, MULTIPLICITY 2 | ON, MULTIPLICITY 2 |
| MPI | mpirun -np 14, NPROC_REP=2 | mpirun -np 14, NPROC_REP=2 |

### 3.3 掺杂体系

| 体系 | 掺杂位点 | 路径 |
|------|----------|------|
| Pristine | — | [001] + [100] |
| Co-doped | Mg替换 | [001] |
| Cu-doped | Mg替换 | [001] |
| Fe-doped | Mg替换 | [001] |

掺杂后需重新弛豫IS和FS（自旋极化态不同）。

### 3.4 分析内容

- 迁移能垒（最大能量差值）
- Bader电荷分析（迁移过程中K⁺电荷变化）
- COHP/COBI成键分析（过渡态K-O键特征）
- 掺杂对能垒的影响机制（d电子工程观点）
